Data Binding

Le Binding :

Le data binding permet de synchroniser les données entre le Model et la Vue dans l'architecture MVC.


- Plusieurs façons pour binder une vue :
  • ng-bind="firstname"
  • { {expression} }
  • ng-model="firstname"

Le Controller :

C'est graçe aux objets javascript, déclarés dans le controller, qu'on peut allimenter la vue de otre application.
la directive ng-controller définie le controller de l'application.

$scope :

AngularJS invoque le controller avec l'objet $scope.
On peut déclarer le controller dans un fichier externe, puis l'inclure dans la vue, à la fin du fichier.
On peut mettre dans le $scope, par exemple, un tableau puis l'afficher au niveau de la vue en utilisant :
  • ng-repeat="x in names"

$Rootscope :

Une variable Scope est accessible depuis la vue de MyApp1 (par exemple).
Une variable Rootscope est accessible depuis toute les application (MyApp1, MyApp2 ...)
